Google OAuth Setup

  1. Visit Google Cloud Console → Credentials
  2. Create a new project or select an existing one
  3. Go to "OAuth consent screen" → Choose "External" → Configure required fields
  4. Add scopes: email and profile
  5. Create Credentials → "OAuth client ID" → "Web application"
  6. Configure authorized origins and redirect URIs (see above)
  7. Copy the Client ID and Client Secret to Step 2
